# NHL Game Calendar
A full-stack web application that displays NHL schedules in an interactive monthly calendar. Users can filter games by season and team, navigate through months, and view detailed game information.

---
## Live Demo
https://nhl-game-calendar.onrender.com

---
## Features

- Interactive monthly calendar view of NHL games
- Filter by season and team
- Dynamic month navigation based on real NHL schedule data (including playoffs)
- Game details modal (teams, venue, time)
- Automatically selects current NHL season and month
- Fully deployed web application
- Currently features last 6 seasons, planning to add more
